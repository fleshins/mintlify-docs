This release delivers **faster onboarding**, **clearer setup guidance**, **better conversation readability**, and **more reliable documentation discovery** so teams can get into Brixo more quickly and work with less friction.

Below are the most important changes in this release (not every internal tweak, just the ones we think deliver the most value).

---

## 🚀 **Faster onboarding**

- **Quicker move into the product:** After a conversation import finishes during setup, users are now taken straight to the dashboard and shown a clear success message. — New users can start exploring Brixo right away instead of wondering what to do next.
- **Welcome email after setup:** New customers now receive a welcome email when onboarding is complete. — It gives users a clear path for next steps and helps them get value sooner.

## 🧭 **Clearer setup guidance**

- **Simpler setup language:** Setup and onboarding screens now explain documentation and conversation imports more clearly. — Users can better understand what Brixo needs and complete setup with less confusion.
- **Easier documentation review:** The documentation review step is now easier to read while users confirm the content Brixo should learn from. — It is easier to verify the right content before moving forward.

## 💬 **Better conversation reading**

- **Formatted messages display more naturally:** Conversation messages now render rich content such as links, lists, tables, and code blocks in a cleaner way. — Users can read and understand conversations more easily, especially when messages include structured content.
- **Plain text view remains available:** Users can turn formatted messages off in preferences and return to the original plain text view. — Teams can choose the reading style that works best for their workflow.

## 🛠️ **More reliable documentation discovery**

- **Improved documentation scans:** Website documentation checks are now more forgiving when scanning for docs. — Brixo is more likely to find the right documentation automatically on the first try.

Thanks for using Brixo! We hope these updates make your day-to-day workflows smoother, faster, and more intuitive.
