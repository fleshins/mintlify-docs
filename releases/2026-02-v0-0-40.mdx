This release delivers **guided documentation setup**, **smarter conversation analysis**, **clearer import visibility**, and **improvements to Explorer workflows** to make it easier to get value from your customer conversations.

Below are the most important changes in this release (not every internal tweak, just the ones we think deliver the most value).

---

## 📚 **Guided product documentation setup**

- **Step-by-step documentation onboarding:** You can now scan your docs site, review discovered pages, add additional URLs or files, and confirm the right sources in one guided flow — making it easier to set up Brixo with the documentation that matters.
- **Automatic filtering of non-documentation pages:** Pages that appear to be marketing or legal content are automatically hidden during scans, while still allowing you to review and re-include them — keeping your knowledge base focused on product documentation.
- **Documentation preferences remembered for future scans:** Brixo remembers which pages you include or exclude when reviewing scanned docs — reducing manual cleanup when you run future scans.

## 🧠 **More accurate conversation insights**

- **Conversation analysis learns from your approved docs:** Conversation analysis now uses your approved documentation alongside recent conversations to better understand your product language — producing more relevant topic and intent insights.

## 📥 **Clearer data import progress**

- **Separate stages for importing and analyzing:** Import jobs now show distinct phases for importing and analyzing — helping you understand exactly what stage your data is in.
- **More accurate Import History progress counts:** Import History now displays cleaner, more reliable progress totals — making it easier to see how much data has been processed.
- **Required fields prioritized during import mapping:** Required fields now appear first when mapping import data — helping you complete setup faster.

## 🔎 **Explorer workflow improvements**

- **Automatic filter name suggestions:** When saving a filter in Explorer, Brixo can now suggest a filter name automatically — making it faster to organize reusable views.
- **More accurate conversation totals with filters:** Conversation totals remain accurate even when filters are applied — so counts and sorting better reflect the true size of each conversation.

## ✨ **Smoother workspace experience**

- **Improved scrolling behavior:** Scrolling has been refined across key workspace areas — making longer pages easier to navigate.
- **Refreshed sign-in and sign-up screens:** Authentication pages now feature updated visuals and clearer branding — creating a cleaner entry experience.

Thanks for using Brixo! We hope these updates make your day-to-day workflows smoother, faster, and more intuitive.
