This release delivers **clearer navigation cues**, **more consistent naming**, **simpler sign-in wording**, and **a more cohesive import workflow**, alongside **faster background processing** for busy workloads.

Below are the most important changes in this release (not every internal tweak, just the ones we think deliver the most value).

---

## 🧭 **Find your place faster**

- **More consistent page names:** Page names are now aligned across the dashboard, explorer, onboarding, and account areas — You can navigate with more confidence because the same area is labeled the same way everywhere.
- **Clearer browser tab titles:** Browser tab titles are more descriptive and consistent — It’s easier to keep track of your work when you have multiple Brixo tabs open.

## 🔐 **Clearer sign-in and sign-up**

- **Simplified sign-in and sign-up labels:** Authentication pages now use simpler, more consistent wording — Teams can sign in or create accounts with fewer hesitations and fewer mis-clicks.
- **“Magic Link” is now “sign in with email”:** Magic Link sign-in is now described as signing in with email — The option is easier to understand at a glance, especially for new users.

## 📥 **More consistent import workflow**

- **Unified import naming across pages:** Import-related pages now use the same naming throughout the experience — It’s clearer that starting an import and reviewing import history are part of one connected workflow.

## ⚡ **Faster and steadier background work**

- **Higher capacity for background processing:** Capacity was increased for key background jobs (including analytics, telemetry, organization updates, ingestion, and knowledge tasks) — Large or busy workloads move through the system faster, reducing delays for work that depends on background processing.
- **Background workers run with intended performance settings:** An issue was fixed where some background workers could run without the correct performance configuration — The app stays more stable during heavy usage and background tasks remain responsive under load.

Thanks for using Brixo! We hope these updates make your day-to-day workflows smoother, faster, and more intuitive.
