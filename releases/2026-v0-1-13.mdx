This release delivers a new **Insights experience**, **automatic insight generation**, **clearer progress while analysis runs**, **faster feedback review**, and **more consistent reporting**.

Below are the most important changes in this release (not every internal tweak, just the ones we think deliver the most value).

---

## 🔍 **New Insights experience**

- **Insights in one place:** A new Insights view brings important patterns across conversations into a single workspace. — Teams can review key findings faster without piecing them together across multiple conversations.
- **Automatic insight generation:** Insights now build automatically as soon as enough data is ready. — You can move from raw conversation data to useful takeaways with less manual effort.
- **Richer insight details:** Each insight now includes a summary, impact counts, supporting examples, and suggested next steps. — It is easier to understand what is happening, how much it matters, and what action to take.
- **Source conversation access:** You can open the original conversation directly from an insight. — This gives teams the context they need to validate findings and investigate further.

## ⏳ **Clearer analysis progress and recovery**

- **Visible progress while Insights load:** The Insights page now shows progress updates while analysis is running. — Users can see that work is in progress instead of wondering whether analysis has stalled.
- **Helpful guidance when data is limited:** If there is not enough conversation data or the current filter is too narrow, the page explains what to do next. — Teams can fix setup or filtering issues more quickly and get to results sooner.
- **Retry after errors:** You can retry analysis directly from the Insights page if something goes wrong. — This reduces interruptions and makes it easier to recover without leaving the workflow.

## 💬 **Faster feedback review**

- **Feedback shown in conversations:** Thumbs up and thumbs down feedback now appears directly inside conversation views. — Reviewers can immediately connect feedback to the exact response it refers to.
- **Feedback navigation shortcuts:** You can move through feedback events from the conversation navigation area. — This makes feedback review quicker, especially when working through longer conversations.

## 🎯 **More useful filtering**

- **Filter by feedback sentiment:** Explorer now supports filtering conversations by positive or negative feedback. — Teams can quickly focus on successful experiences or investigate problem areas.
- **Cleaner saved filter behavior:** Feedback selections now carry through more consistently in saved filters and active filter chips. — It is easier to reuse important views without having to rebuild filters each time.

## 📊 **More consistent reporting over time**

- **Dashboard respects active time range:** The Dashboard now follows the time range from your active filter. — Reports stay aligned across views, making comparisons and analysis more reliable.
- **Time ranges saved more reliably:** Saved filters now remember selected time ranges more consistently, including longer lookback windows. — You are more likely to see the same results when returning to saved views later.

Thanks for using Brixo! We hope these updates make your day-to-day workflows smoother, faster, and more intuitive.
