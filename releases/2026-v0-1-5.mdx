This release delivers **clearer documentation scan feedback**, **smoother source review**, **better onboarding visibility**, and **improved setup suggestions**. It also includes a small but meaningful polish update to make the installed app feel more complete.

Below are the most important changes in this release (not every internal tweak, just the ones we think deliver the most value).

---

## 📘 **Easier documentation review**

- **Clearer scan outcomes:** Product documentation scans now give clearer guidance when a scan finishes successfully, finds no useful content, or runs into an error — teams can understand what happened faster and know what to do next.
- **Smoother add-more flow:** Adding extra documentation by URL or file is now easier, with the add-more section kept in view and newly added items automatically scrolled into view — you can confirm added sources more quickly and keep moving through setup.
- **Better review of excluded sources:** Sources that were set aside are now easier to review without losing your place — this gives you more control over what gets included with less disruption.

## 🚀 **Smoother onboarding imports**

- **Import History available during setup:** Import History is now available again during onboarding — you can track progress and revisit earlier imports without leaving the setup flow.
- **Refresh import progress faster:** The import progress banner now offers a page reload action instead of a dismiss option — it is easier to refresh and immediately see the latest import results.

## 🤖 **Improved setup suggestions**

- **Upgraded AI model for scanning:** Document scanning and related setup suggestions now use a newer AI model — you should see better documentation discovery and more responsive setup guidance.

## 🏷️ **More polished app experience**

- **Correct app name when installed:** When the app is saved or installed from a browser, it now shows the correct product name instead of a generic label — the installed app feels more polished and is easier to recognize.

Thanks for using Brixo! We hope these updates make your day-to-day workflows smoother, faster, and more intuitive.
