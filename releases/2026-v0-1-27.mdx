This release delivers **longer-term trend visibility**, **faster dashboard performance**, **easier drill-down analysis**, **more reliable insights**, and **clearer import progress**.

Below are the most important changes in this release (not every internal tweak, just the ones we think deliver the most value).

---

## 📈 **Broader trend visibility**

- **1-year chart view:** Key charts on the dashboard and profile pages can now switch to a 1-year view — helping teams spot longer-term patterns instead of only recent changes.
- **12-month default range:** When last-year data is available, pages that show all data now default to the past 12 months — giving you broader context right away.
- **Synced profile time ranges:** User and account profile pages now follow the page’s selected time range automatically — reducing manual controls and keeping charts consistent.

## 🔎 **Easier exploration and drill-down**

- **Expanded dashboard tables:** Topic and intent tables on the dashboard can now expand to show the full list of results — making it easier to explore beyond the top few entries.
- **More reliable drill-down links:** Links from dashboard cards, charts, and tables into deeper analysis views are now more dependable — so you can move from summary metrics to details with less friction.
- **Preserved filter context:** Moving from insights into detailed views now keeps your current filter context — so you do not need to reapply filters after clicking through.
- **More consistent combined filters:** Combined filters now behave more consistently across views — making counts and insights easier to trust as you move around the product.

## ⚡ **Faster and clearer dashboard experience**

- **Faster dashboard loading:** The dashboard now opens and refreshes more quickly, with key page elements appearing sooner — so you can start reviewing results faster.
- **Clearer loading states:** Loading states on the dashboard have been improved while data finishes loading — making it easier to understand what is ready and what is still updating.
- **Better performance for heavy views:** Dashboard charts and summary views have been optimized, especially for longer date ranges — helping larger views feel more responsive.

## 💡 **More reliable insights workflow**

- **Daily insight refresh:** Insights now refresh automatically every day — keeping summaries more current without waiting for a manual refresh.
- **Clear refresh progress and errors:** When insights are refreshing, the page now shows a clear progress state, and failed refreshes show a normal error state instead of outdated results — reducing confusion about what is current.
- **Stronger reliability during busy periods:** Insight generation is more resilient when the system is under heavier load — helping keep results available when demand is high.
- **Better evidence review:** Opening supporting conversations from Insights now keeps the main page in place and scrolls only inside the conversation panel — so you can review evidence without losing your place.

## 🧭 **More trustworthy navigation and progress feedback**

- **More stable dashboard navigation:** Navigation from the dashboard is more stable, reducing cases where the app could jump back to the wrong page — making workflows smoother and easier to follow.
- **More accurate import progress:** Import progress now reflects the actual rows analyzed from source data, including incomplete or one-sided conversations — making progress indicators more trustworthy.
- **Clearer import status messages:** Status messages during imports have been updated to better reflect what is happening — so it is easier to understand how far along a job is.
- **Cleaner in-product messaging:** Small copy updates make “coming soon” messaging clearer and less distracting — improving overall clarity across the experience.

Thanks for using Brixo! We hope these updates make your day-to-day workflows smoother, faster, and more intuitive.
